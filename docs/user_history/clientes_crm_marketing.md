# Clientes, CRM e Marketing

## US02 - Criar conta
**Como** cliente,  
**quero** criar uma conta dentro do site da loja virtual,  
**para que** eu possa fazer compras de forma mais rápida com meus dados cadastrados.

### Regras de negócio
- Os campos obrigatórios para criação da conta são: e-mail e senha.
- Os campos opcionais são: CPF, telefone, CEP e endereço.

## US11 - Cadastrar clientes
**Como** caixa,  
**quero** registrar o cadastro de um cliente com nome, e-mail e telefone,  
**para que** seja construído um perfil de cliente da empresa para campanhas futuras.

## US12 - Exportar base de clientes para tráfego pago
**Como** administrador,  
**quero** extrair os dados dos clientes em formato CSV,  
**para que** eu possa criar públicos personalizados e otimizar campanhas.

## US13 - Gerenciar cupons de desconto
**Como** administrador,  
**quero** adicionar cupons, definir taxa de desconto e associar produtos ao cupom,  
**para que** eu tenha controle sobre os descontos da loja.

## US14 - Gerenciar cupons de desconto para parceiros
**Como** administrador,  
**quero** associar cupons a parceiros ao gerar o cupom,  
**para que** eu possa rastrear a origem das vendas e bonificar parceiros.
