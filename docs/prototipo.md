# Documentação de Prototipação - Figma

## Visão Geral

A prototipação de alto nível do projeto foi realizada utilizando **Figma**, plataforma líder de design e prototipagem colaborativa. O protótipo serve como validação visual dos fluxos de usuário e padrões de interface que guiarão a implementação do sistema.

Este documento centraliza os links dos protótipos finalizados para acesso e revisão.
